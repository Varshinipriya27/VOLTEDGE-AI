# VoltEdge AI

VoltEdge AI is a futuristic AI-powered Electronics Learning Platform for engineering students. It includes a protected login flow, interactive 3D component explorer, virtual electronics lab, oscilloscope visualizer, quiz arena, AI tutor, and progress analytics dashboard.

## Features

- Student/faculty login modal with demo credentials
- Protected learning modules after login
- Three.js 3D electronics component explorer
- Drag-and-drop virtual lab simulations
- Oscilloscope and signal visualizer
- Quiz arena with feedback and performance tracking
- AI tutor knowledge base for electronics concepts
- Personalized progress and readiness analytics
